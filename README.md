GeoToolsを使ったGISプログラミング 
================================

Java用のオープンソースGISライブラリのGeoToolsを使ったプログラミング技術習得を行います。

GeoTools: http://www.geotools.org/

ディレクトリ構成(Directory tree)
-------------------------------

* ShapefileViewer: シェープファイルを指定し地図表示するプログラム
* mapdata: 地図データを置くトップレベルディレクトリ

プログラム開発環境
-----------------

JDK 7u10
NetBeans 7.3
GeoTools 8.4

プログラム内容
-------------

* ShapefileViewer
GeoToolsドキュメントのQuickStartに記載のものとほぼ同等



地図データ内容
-------------

* 国土交通省 国土政策局 国土数値情報 海岸線データ(すべての県を1つにまとめたもの)
